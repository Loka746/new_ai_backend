# VibeCoding Project Platform

A complete full-stack project management platform built with FastAPI and Vanilla JS.

## Setup Instructions

1. Create a virtual environment:
   `python -m venv venv`
2. Activate the virtual environment:
   - Windows: `venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
3. Install dependencies:
   `pip install -r requirements.txt`
4. Copy `.env.example` to `.env` and update variables if needed.
5. Run the application:
   `uvicorn main:app --reload`
6. Open your browser and navigate to `http://localhost:8000/static/index.html`
